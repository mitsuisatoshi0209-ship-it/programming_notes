# Programming Reference

このリポジトリは、コーディング作業を快適に進めるための備忘録をまとめるための場所です。

## 目的

- コーディング作業でよく使う手順やコマンドを記録する
- VS Code や拡張機能の使い方をまとめる
- Git / GitHub の基本操作を自分用のメモとして残す
- AI ツールや API の使い方を整理する
- いざ必要になったときにすぐ見返せるようにする

## 内容

- programming_note1.md: プログラミング言語の構造と選び方
- programming_note2.md: VS Code と実行環境の入れ方
- programming_note3.md: Git / GitHub / Continue / API の基礎
- memo.md: 簡単にまとめたプログラミングに関する情報まとめ

## 使い方

このリポジトリは、自分のための備忘録として使います。
必要なときに各 Markdown を開き、手順やコマンドを確認してください。

## 備考

内容は実際に使いながら随時更新します。
特に、すぐ忘れてしまうコマンドや設定、便利ツールの使い方を中心にまとめていきます。
