# xsel-shortcut-setup

Linux環境（Ubuntu / RedHat系）で、クリップボードの操作を快適にする `xsel` のインストールとエイリアス（`ccc` / `vvv`）の自動設定スクリプトです。

この設定を導入すると、ターミナル上で以下のような直感的な操作が可能になります。
- `ccc [ファイル名]` または `[コマンド] | ccc` : クリップボードにコピー
- `vvv` または `vvv > [ファイル名]` : クリップボードから貼り付け（出力）

---

## 🚀 使い方

自分の環境に合わせたスクリプトを1行実行するだけで、インストールから `.bashrc` へのエイリアス追記、反映までが自動で行われます。

### 1. 自分のパソコン用 (Ubuntu / sudo 権限あり)
ローカルのUbuntu環境などで、管理者権限を使ってシステム全体にインストールする場合に使用します。

```bash
bash xsel_setup_ubuntu.sh

```

### 2. ワークステーション用 (RedHat系 / sudo 権限なし)

大学のワークステーションなど、`sudo` 権限がない環境で、RPMパッケージからバイナリ（`xsel`）を抽出して `~/bin/` に配置して利用する場合に使用します。

```bash
bash xsel_setup_ws.sh

```

---

## 💡 仕組み（登録されるエイリアス）

設定が完了すると、`~/.bashrc` に以下の関数とエイリアスが追加されます。

* **`ccc`** (`_ccc_copy`): 引数にファイルを指定した場合はその中身を、パイプ（`|`）で渡された場合はその標準入力を自動で判別してクリップボードに送ります。`ccc *` でカレントディレクトリの全ファイルをまとめてコピーすることも可能です。
* **`vvv`**: クリップボードの中身を標準出力に吐き出します。`vvv > output.txt` でファイルへ上書き保存、`vvv >> output.txt` でファイル末尾への追記ができます。

```
